# 🌐 Portafolio Profesional – Emmanuel Bravo Campos

Sitio web de portafolio personal desarrollado con **HTML, CSS y JavaScript vanilla**.

🔗 **[Ver en vivo](https://emmanuelbravoc.github.io)**

---

## 📁 Estructura del proyecto

```
EmmanuelBravoC.github.io/
├── index.html   → Estructura y contenido HTML
├── styles.css   → Estilos, variables CSS, animaciones y diseño responsivo
├── main.js      → Datos, renderizado dinámico del DOM e Intersection Observer
└── README.md    → Este archivo
```

## ✨ Características

- Diseño responsivo (mobile-first)
- Animaciones con CSS y Intersection Observer API
- Navegación activa con scroll
- Secciones: Sobre mí, Habilidades, Experiencia y Certificaciones
- Renderizado dinámico con JavaScript puro (sin frameworks)

## 🛠️ Tecnologías

- HTML5 semántico
- CSS3 (variables, flexbox, grid, animaciones)
- JavaScript ES6+ (DOM manipulation, IntersectionObserver, arrow functions, template literals)
